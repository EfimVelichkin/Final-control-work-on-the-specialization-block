# Задание №1
### Используя команду cat в терминале операционной системы Linux, создать два файла Домашние животные (заполнив файл собаками, кошками, хомяками) и Вьючные животными заполнив файл Лошадьми, верблюдами и ослы), а затем объединить их. Просмотреть содержимое созданного файла. Переименовать файл, дав ему новое имя (Друзья человека).
---
echo "собаки, кошки, хомяки" > "Домашние животные"\n
echo "Лошади, верблюды, ослы" > "Вьючные животными"\n
cat "Домашние животные" "Вьючные животными" > "Объединенный файл"\n
cat "Объединенный файл"\n
mv "Объединенный файл" "Друзья человека"
---
